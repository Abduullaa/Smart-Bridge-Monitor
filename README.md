# Smart Bridge Health Monitor
![Dashboard](dashboard.png)
A structural health monitoring (SHM) simulation built in Python.

## What it does
- Simulates real-time sensor data from a bridge (vibration, strain, temperature)
- Detects anomalies and fires alerts when readings exceed engineering thresholds
- Logs all data to CSV for analysis
- Generates charts showing sensor behaviour over time

## Built with
- Python, NumPy, Pandas, Matplotlib

## Background
Built as a self-development project combining civil engineering knowledge 
with data skills — targeting smart infrastructure and digital twin applications 
in UAE Vision 2031 projects.
